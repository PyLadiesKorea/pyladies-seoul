# 🚀 이미지 업로드 퀵 가이드

PyLadies Seoul 웹사이트에 이미지를 추가하는 간단한 방법을 알려드립니다!

## 📸 1단계: 이미지 업로드

1. **관리자 로그인** → http://localhost:8000/admin/
2. **"Images"** 메뉴 클릭
3. **"Add Image"** 버튼 클릭
4. 이미지 파일을 **드래그앤드롭** 또는 **파일 선택**
5. **제목** 입력 (예: "2024 파이썬 워크샵")
6. **Alt text** 입력 (예: "참가자들이 노트북으로 코딩하는 모습")
7. **저장** 클릭

## 🎨 2단계: 페이지에 이미지 추가

### 옵션 1: 큰 이미지 하나만
1. 페이지 편집 → **Body** 섹션
2. **"Full Width Image"** 선택
3. 업로드한 이미지 선택
4. 캡션 추가 (선택사항)

### 옵션 2: 이미지 + 설명 텍스트  
1. **"Image with Text"** 선택
2. 이미지 선택 + 위치 선택 (좌측/우측)
3. 제목과 설명 텍스트 작성

### 옵션 3: 여러 이미지 갤러리
1. **"Image Gallery"** 선택
2. **"Add Image"** 버튼으로 이미지 추가
3. 각 이미지에 캡션 작성

## ✅ 팁 & 권장사항

### 이미지 준비할 때
- **크기:** 최대 10MB
- **형식:** JPG, PNG, WebP, GIF
- **해상도:** 1200x800px 정도가 적당

### Alt Text 작성할 때  
```
✅ 좋은 예: "Jane Doe가 Python 코드를 설명하는 모습"
❌ 나쁜 예: "이미지", "사진", "IMG_001.jpg"
```

### 언제 어떤 블록을 사용할까?
- **단일 중요 이미지** → Full Width Image
- **설명이 필요한 이미지** → Image with Text  
- **여러 사진 모음** → Image Gallery

## 🔧 문제 해결

### 이미지가 업로드 안될 때
- 파일 크기 확인 (10MB 이하)
- 파일 형식 확인 (JPG, PNG, WebP, GIF만)
- 파일명에 한글이나 특수문자 있으면 영문으로 변경

### 이미지가 안 보일 때
- 브라우저 새로고침 (Ctrl + F5)
- 다른 브라우저에서 확인
- 이미지가 올바르게 선택되었는지 확인

## 🎯 자주 사용하는 패턴

### 이벤트 후기 페이지
```
1. Full Width Image (메인 단체 사진)
2. Image with Text (발표자 소개 + 사진)  
3. Image Gallery (현장 사진들)
4. Image with Text (참가자 후기 + 사진)
```

### 멤버 소개 페이지
```
1. Image with Text (프로필 사진 + 자기소개)
2. Image Gallery (작업한 프로젝트 스크린샷들)
```

### 워크샵 안내 페이지  
```
1. Full Width Image (워크샵 포스터)
2. Image with Text (강사 소개 + 사진)
3. Image Gallery (이전 워크샵 현장 사진들)
```
